Datalist Cars

  • La etiqueta datalist especifica una lista de opciones predefinidas para un elemento input.
  • La etiqueta datalist se utiliza para proporcionar una función de "autocompletar" para los elementos input. Los usuarios verán una lista desplegable de opciones predefinidas a medida que ingresan datos.
  • El atributo id del elemento datalist debe ser igual al atributo list del elemento input (esto los une).

The input multiple

  • El atributo múltiple es un atributo booleano.
  • Cuando está presente, especifica que el usuario puede ingresar más de un valor en el elemento input.

Consejo: Para [input type = "file"]: para seleccionar varios archivos, mantenga presionada la tecla CTRL o SHIFT mientras selecciona.

Nota: El atributo múltiple funciona con los siguientes tipos de entrada: correo electrónico y archivo.

The autofocus attribute

La propiedad de enfoque automático es un booleano que refleja el atributo HTML de enfoque automático, que indica si el elemento asociado obtendrá el foco de entrada cuando se cargue la página, a menos que el usuario lo anule.
